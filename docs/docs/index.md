# Ella_PhD_NLP_project documentation!

## Description


This repository contains a Python-based NLP pipeline for processing:

- Audio recordings
- Speaker diarization outputs
- Transcripts (.docx or .txt)

The pipeline produces:
- Patient-only audio
- Cleaned transcripts
- Structured feature tables per patient

The project follows a strict data workflow:

**RAW → INTERIM → PROCESSED → REPORTS**

and is developed and tested using **Conda**.

---

## Main Components

- Audio preprocessing using speaker diarization
- Transcript cleaning and normalization
- Feature extraction and dataframe generation
